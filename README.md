# 灵阁 LingLoft 团队官网

这是灵阁团队的官方网站，基于 Nuxt 3 开发，采用战术 UI 风格设计，展示团队信息、成员、项目及联系方式。

## 来源声明

本项目魔改自 [主核Kernyr的个人网站](https://github.com/kernel4632/website)。

根据 **GPL-3.0** 协议要求，本项目已开源并保留原作者信息。感谢原作者的优秀设计与代码贡献。

## 项目特点

- 🚀 **Nuxt 3**: 基于最新的 Nuxt 3 框架构建，性能卓越。
- 🎨 **战术 UI**: 独特的战术风格界面设计，极具科技感。
- 📱 **响应式设计**: 完美适配各种屏幕尺寸。
- 🛠️ **易于定制**: 通过 `public/data.json` 即可快速修改网站内容。

## 快速开始

### 第一步：安装依赖

在项目根目录下运行以下命令安装依赖：

```bash
npm install
```

### 第二步：启动开发服务器

启动本地开发服务器，预览实时效果：

```bash
npm run dev
```

访问 [http://localhost:3000](http://localhost:3000) 即可查看。

### 第三步：构建生产版本

构建用于部署的生产版本：

```bash
npm run build
```

### 第四步：预览生产版本

在本地预览构建后的生产版本：

```bash
npm run preview
```
